## 🛠️ Tech Stack

### Frontend
- ⚛️ React 19 with TypeScript
- 🎨 HTML5 & Tailwind CSS
- 🚦 TanStack Router & Query
- 🧩 shadcn/ui components
- 📦 Vite
- 🔍 Zod validation
- ✨ ESLint

### Backend
- 🚀 Express.js with TypeScript
- 🗄️ PostgreSQL
- 🔄 Redis
- 🔌 Socket.io for real-time communication
- ☁️ Cloudinary for media storage

### Development Tools
- 📦 pnpm
- 📦 npm
- 🎨 Figma
- 🧪 TypeScript
- 🔍 ESLint
